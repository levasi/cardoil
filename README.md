# Cardoil Avantaj — Website Redesign

Modern redesign of [cardoilavantaj.ro](https://cardoilavantaj.ro/) built with Next.js 16, Tailwind CSS v4, and shadcn/ui.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

| Route | Description |
|-------|-------------|
| `/` | Homepage — hero, services, testimonials, offer form |
| `/despre` | About the company |
| `/servicii` | Detailed services |
| `/oferta` | Request a quote |
| `/contact` | Contact information and form |

## Tech stack

- **Next.js 16** (App Router, Turbopack)
- **Tailwind CSS v4**
- **shadcn/ui** (Base UI)
- **TypeScript**

## Deploy

Deploy to [Vercel](https://vercel.com) or run:

```bash
npm run build
npm start
```
